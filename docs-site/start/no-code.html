Начинаю с нуля — кода ещё нет

Сначала заводим карту гипотез. Код эксперимента появится позже в <репо>/projectcode/.

Одной командой

  1. Клонируйте и запустите ResearcherOS 
    git clone git@github.com:ZoyaV/ResearcherOS.git ReseachOS
cd ReseachOS
./scripts/koi-serve.sh start
  2. Создайте пустой проект 
    python -m koi.projects.install_cli install my_idea \
  --create --title "Моя исследовательская проблема"
./scripts/koi-serve.sh restart
# http://127.0.0.1:8080

    На диске появится:

    tree/my_idea/koi-structure/   ← project.md, research.json
my_idea/projectcode/          ← место под код

В интерфейсе

  1. Откройте http://127.0.0.1:8080, в списке проектов — + Новый проект: название, описание, короткий латинский ярлык и программу.
  2. На карте появится проблема — добавляйте причины, гипотезы и методы через «+».

С помощником в редакторе

Промпт:

Создай новый проект ResearcherOS с нуля.

1. Спроси меня о проблеме, области и том, какие эксперименты реалистичны.
2. Предложи дерево: проблема → причины → гипотезы → методы.
3. Создай через
   python -m koi.projects.install_cli install <ярлык> --create
   (tree/<ярлык>/koi-structure + <ярлык>/projectcode).
4. Пиши понятным языком. Не коммить без моей явной просьбы.
