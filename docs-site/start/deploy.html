Уже есть ResearcherOS-проект

Дерево гипотез уже существует (у вас или у команды). Осталось правильно разложить файлы на диске и запустить интерфейс. Выберите свой случай. 

workspace/
├── ReseachOS/                      ← движок
├── tree/
│   └── <репо>/koi-structure/       ← ветка koi/research
└── <репо>/                         ← код, любая ветка

Сервер заходит в папку tree и на следующем уровне ищет */koi-structure/project.md.

Первый запуск на этой машине

Ветка koi/research уже есть на remote. Нужно клонировать движок и code-репо, повесить research-ветку в tree/ и открыть UI.

  1. Клон движка и code-репо рядом 
    git clone git@github.com:ZoyaV/ResearcherOS.git ReseachOS
git clone <url-вашего-code-репо>
# оба каталога — siblings, имена папок любые
# workspace/ReseachOS/
# workspace/<репо>/
  2. Повесить koi/research в tree/ 
    cd ReseachOS
./scripts/koi-serve.sh start
python -m koi.projects.install_cli install <репо>
# имя папки code-репо, как на диске
python -m koi.projects.install_cli status

    Появится tree/<репо>/ — git worktree на ветке koi/research с koi-structure/ внутри.

  3. Открыть интерфейс 
    ./scripts/koi-serve.sh restart
# http://127.0.0.1:8080
# API: http://127.0.0.1:8010/docs

Миграция со старого layout

Материалы исследования уже локально, но не в tree/: лежат в <репо>/koi-structure/ или в <репо>/.koi-sync-worktree/. Code-репо и движок уже рядом.

  1. Проверить, что CLI видит migrate 
    cd ReseachOS
python -m koi.projects.install_cli status
# у нужного <репо> case: migrate_to_tree
  2. Перенести в tree/ 
    python -m koi.projects.install_cli migrate <репо>
# то же, что install — кейс определится сам
./scripts/koi-serve.sh restart

    Старый checkout в code-репо можно удалить после проверки, что tree/<репо>/koi-structure/project.md на месте и проект виден в UI. На code-ветке пути остаются в .gitignore.

Дальше синхронизация: кнопка Sync в UI или python -m koi.projects.sync_cli pull|push --project-id <id>. Если дерева гипотез ещё нет вообще — путь «впервые подключаю свой код» (промпт агенту).
