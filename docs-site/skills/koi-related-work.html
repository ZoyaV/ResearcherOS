Related Work

koi-related-work

Кнопка Related Work

Из очереди интерфейса литературы написать раздел Related Works и вернуть его на страницу.

Что делает

Обрабатывает задачу из очереди Related Work: сразу принимает её, забирает контекст с кластерами статей, пишет обзор только по переданным фактам и сохраняет ответ обратно в интерфейс. Без принятия страница «ждёт запрос»; без сохранения ответа черновик не появится.

Зачем

Чтобы обзор литературы рождался из выбранных в интерфейсе кластеров и сразу попадал на страницу Related Work, без ручного копирования туда-сюда.

Когда вызывать

  • В интерфейсе нажали Related Work и задача попала в очередь
  • Постоянный чат Literature Inbox получил сигнал о новой задаче
  • Нужно дописать или пересохранить ответ по уже принятой задаче

Как вызвать

Попросите проверить очередь Related Work и обработать задачу по цепочке: принять → контекст → текст → сохранить. Удобно держать отдельный постоянный чат под литературу.

Пример фразы AI-помощнику в редакторе:

Проверь очередь Related Work и обработай задачу:
принять → забрать контекст → написать обзор → сохранить ответ.

Схема работы

Схема — линейный конвейер очереди. Каждый шаг обязателен: без «принять» интерфейс не перейдёт в работу, без «сохранить ответ» текст останется только у AI-помощника.

flowchart LR
  A[Очередь из интерфейса] --> B[Принять задачу]
  B --> C[Забрать контекст]
  C --> D[Написать обзор]
  D --> E[Сохранить ответ в страницу]
