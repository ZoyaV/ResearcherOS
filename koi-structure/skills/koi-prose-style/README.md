# Тексты понятным языком

Скилл смотрит **любой текст, который увидит человек** в проекте ResearchOS:
заголовки и описания на дереве, карточки канбана, формулировки вопросов,
нарративы, заметки в базе знаний — и приводит его к живому языку.

Цель простая: новый человек должен понять написанное без глоссария
и без знания внутренних id проекта.

## Зачем он нужен

Агент легко скатывается в смесь английского с русским, голые аббревиатуры
и «технический» тон. Этот скилл ловит такие места и переписывает:
сначала смысл своими словами, потом при необходимости сокращение в скобках
(например «доля кликов по показам (CTR)»). На карте — короткий ярлык,
подробности — в описании.

## Как пользоваться

В установленном ResearchOS скилл включается, когда агент пишет или правит
текст для человека. Агент следует `SKILL.md` в `agents/skills/koi-prose-style/`;
эталонные пары «хорошо / плохо» — в `examples.md`.

## Состав пакета

- `SKILL.md` — правила и цикл ревью для агента
- `examples.md` — примеры удачных и слабых формулировок
- `manifest.yaml` — карточка для каталога Hub
