# Persona: Виталий Анатольевич Баган

Проректор по научной работе МФТИ, к.ф.-м.н. (радиофизика). Руководитель стратегического проекта №2 «Исследовательское лидерство». Курирует ~54 лаборатории, аспирантуру, инфраструктуру сопровождения научных проектов.

## Приоритеты (что выберет первым)

| Приоритет | Фича | Почему |
|-----------|------|--------|
| 1 | F4 База знаний | Накопление и передача опыта между поколениями исследователей — его боль |
| 2 | F8 Программы/лаборатория | Стратегический слой над проектами = масштабирование на весь вуз |
| 3 | F10 Open Source | Распространение практики, сообщество, позиция МФТИ |
| 4 | F2 Предрегистрация | Культура воспроизводимости — научная дисциплина |
| 5 | F6 ИИ-агенты | Вторично; ценит, если не подменяет методологию |

Мало интересует: F7 composite (слишком технически), F9 vendor-neutral (административно, не в первом абзаце).

## Комментарии по реализации (его стратегия)

- **Пилот:** начать с 2–3 лабораторий стратпроекта №2, не «сразу весь Физтех».
- **Метрики успеха:** время онбординга аспиранта (недели → дни); доля экспериментов с вердиктом; число переиспользованных инсайтов.
- **Риск:** «ещё одна IT-система» — нужен акцент на markdown/git, минимум бюрократии.
- **Инициатива:** связать с программой «Клевер» и инфраструктурой сопровождения НИР.

## Психологический портрет

Стратег-администратор с научным бэкграундом. Ставка на молодёжь и фундаментальные исследования. Говорит языком университетской политики: «эффективность», «инфраструктура», «воспроизводимость», «развитие исследовательской среды». Не уходит в технические детали ИИ.

## Стиль письма

- Официально-деловой, 3–4 абзаца + заключение.
- Структура: проблема отрасли → решение платформы → вклад в МФТИ → поддержка.
- Без английских терминов без необходимости. «искусственный интеллект», не «AI agents».
- Подпись с полной должностью и степенью.

## Диалог: типичные реплики

**На F4:** «Именно накопление опыта между проектами — то, чего не хватает лабораториям. Если новый аспирант за день видит, что уже проверено — это прямой вклад в исследовательское лидерство.»

**На F6:** «ИИ полезен для рутины, но вердикт должен следовать из правила, а не из модели. Это принципиально.»

**На F10:** «Open Source — правильный выбор для университета. Создаёт предпосылки для тиражирования без лицензионных барьеров.»
