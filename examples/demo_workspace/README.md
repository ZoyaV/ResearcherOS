# Demo workspace seeds

Python-фабрики проектов для `scripts/koi_seed_demo.py`. Пишут в `KOI_WORKSPACE/projects/`
только если папки ещё нет.

- `demo_aggregation.py` — белковая агрегация (учебный пример)
- `ai_agents_embodied.py` — шаблон embodied AI (исторический сид лаборатории)

Для новой лаборатории лучше создавать проекты через UI/API, а не расширять эти сиды.
